Un poco sobre mí

Hola, soy un desarrollador que le encanta aprender cosas nuevas, y nunca me dejo enrredar por cosas que no se, porque si no lo se, lo aprendo.

Mi aventura en el mundo del desarrollo comenzó con curiosidad y se convirtió en pasión. Me fascina cómo unas líneas de código pueden resolver problemas reales y crear experiencias increíbles.

Trabajo principalmente con Python para el backend y JavaScript para crear interfaces que no solo funcionan bien, sino que también se sienten bien. Creo en el código limpio, las buenas prácticas y en aprender algo nuevo cada día.

Cuando no estoy programando, probablemente esté leyendo sobre nuevas tecnologías o cosas impactantes, experimentando con frameworks o pensando en cómo hacer que el mundo sea un poco mejor a través del código.

juego de asteroides con python

Un juego clásico de asteroides desarrollado en Python utilizando la biblioteca Pygame. Controla una nave espacial, evita y destruye asteroides mientras acumulas puntos. Perfecto para aprender conceptos básicos de programación de juegos.

Sistema de Gestión

Aplicación web para la gestión de inventarios que permite agregar, editar y eliminar productos, generar alertas de stock bajo y visualizar reportes en tiempo real. Está construida con HTML, CSS y JavaScript, usando LocalStorage para persistencia de datos y una arquitectura modular fácil de escalar.
